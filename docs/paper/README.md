# Статья (LaTeX)

| Файл | Назначение |
|------|------------|
| `процедурная_мета_рефлексия.tex` | Основная русская версия |
| `article.tex` | Дубликат содержания (англ. аннотация внутри) |
| `figures/pmr_I_LPQ_boxplots.png` | Рисунок для разд. 4 (генерируется скриптом) |

Сборка (нужен XeLaTeX и шрифт с кириллицей, напр. Arial):

```bash
xelatex процедурная_мета_рефлексия.tex
xelatex процедурная_мета_рефлексия.tex   # второй проход для ссылок
```

Рисунок:

```bash
python3 ../../scripts/plot_pmr_article_figures.py
```
